# Netflix-Clone-Task-3

A fully responsive Netflix India Clone Website built using HTML, CSS, and JavaScript.

- **GitHub Repository**: [Netflix-Clone-Task-3](https://github.com/sreesahithikakumani/Netflix-Clone-Task-3)

## 📝 Description

This project is a responsive clone of the Netflix India website. It showcases modern web design principles with a clean, user-friendly interface that works seamlessly across all devices.

## ✨ Features

- Fully responsive design for mobile, tablet, and desktop
- Modern UI/UX similar to Netflix
- Smooth animations and transitions
- Interactive components
- Cross-browser compatible

## 🛠️ Technologies Used

- **HTML5** - Markup structure
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and functionality

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sreesahithikakumani/Netflix-Clone-Task-3.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Netflix-Clone-Task-3
   ```

3. Open `index.html` in your web browser

## 🚀 Usage

Simply open the project in your browser and explore the Netflix clone interface. No additional setup or dependencies required.
